# 🗄️ Banco de Dados | UDF

Este repositório contém o material de estudo, scripts SQL e projetos desenvolvidos na disciplina de **Banco de Dados**, ministrada pelo professor **Raniere Azevedo** no Centro Universitário do Distrito Federal (UDF).

---

## 🎓 Sobre a Disciplina
A disciplina aborda os fundamentos de bancos de dados relacionais, desde a concepção teórica até a implementação prática e manipulação de dados.

### 🧠 Tópicos Estudados
* **Modelagem Conceitual:** Diagramas Entidade-Relacionamento (DER).
* **Modelagem Lógica:** Transformação para o Modelo Relacional.
* **Normalização:** Aplicação das formas normais (1FN, 2FN, 3FN).
* **Linguagem SQL:**
    * **DDL:** `CREATE`, `ALTER`, `DROP`.
    * **DML:** `INSERT`, `UPDATE`, `DELETE`.
    * **DQL:** `SELECT`, `JOINs`, `GROUP BY`, `HAVING`.
* **Objetos do Banco:** Views, Triggers e Stored Procedures.

---

## 🛠️ Tecnologias e Ferramentas
* **SGBD Principal:** MySQL 
* **Modelagem:** MyCompiler / MySQL Workbench
